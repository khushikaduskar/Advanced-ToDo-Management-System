# 📋 Taskify Pro

A modern and responsive **Task Management Web Application** built using **Python Flask** and **MySQL**. Taskify Pro helps users organize, track, and manage daily activities efficiently through a clean dashboard and user-friendly interface.

## 🌟 Overview

Taskify Pro is a full-stack web application designed to improve productivity by allowing users to create, organize, and monitor tasks in one place. The application supports task categorization, priority management, due-date tracking, and completion monitoring.

## ⚙️ Installation Guide

### 1️⃣ Clone the Repository

```bash
git clone https://github.com/your-username/Taskify-Pro.git
cd Taskify-Pro
```

### 2️⃣ Install Required Packages

```bash
pip install -r requirements.txt
```

### 3️⃣ Configure MySQL Database

Create the database and table using:

```sql
CREATE DATABASE todo_db;
```

Then execute the queries available in `database.sql`.

### 4️⃣ Update Database Credentials

Open `app.py` and configure:

```python
host="localhost"
user="root"
password="your_password"
database="todo_db"
```

### 5️⃣ Run the Application

```bash
python app.py
```

### 6️⃣ Open in Browser

```text
http://127.0.0.1:5000
```
